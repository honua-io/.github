# Honua

Honua is a geospatial server that exposes one shared geospatial capability set through
many protocol adapters — GeoServices REST, OGC API, classic OGC services (WFS/WMS/WMTS),
OData, STAC, MVT/TileJSON, COG/raster, MCP, and gRPC.

## Security

We welcome security research. See our [Security Policy](https://github.com/honua-io/.github/blob/main/SECURITY.md)
and report vulnerabilities to **security@honua.io**.

Our security posture, certifications, and self-attestations are published on our
trust center (see honua.io).
